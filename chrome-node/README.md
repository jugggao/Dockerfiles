# Node 使用谷歌浏览器插件推流

1. 安装 Chrome，由于需要安装谷歌浏览器插件，因此无法使用 Headless Chrome。
2. 安装 X Server 及 Fluxbox 支持图形化界面支持。
3. 安装 Node 与 FFMpeg 支持推流程序的运行。
4. 安装 VNC 支持 调试。
